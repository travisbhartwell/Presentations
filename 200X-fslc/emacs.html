What is it?

  • A text editor?
  • An OS?
  • A religion?
  • I think the answer is probably "Yes!" to all three.

Okay, the serious answer

  • To quote the Emacs Manual:
  • Emacs is the
    • extensible,
    • customizable,
    • self-documenting real-time display editor.
  • Emacs is written in, customized with, and extended with the Emacs Lisp Language

History of Emacs

  • 1974: Began as a set of macros for the TECO editor on the ITS OS.
  • 1976: Richard Stallman organized all of these macros ("Editor MACroS") into a single command set.
  • 1978: Bernard Greenberg wrote Multics Emacs, using a Lisp as an extension language. That proved to make Emacs more extensible than ever before, so all subsequent Emacsen used Lisp.
  • 1981: James Gosling wrote the first Unix Emacs.

History of Emacs (cont.)

  • 1984: Richard Stallman began writing Gnu Emacs, now the defacto standard Emacs
  • 1991: Jamie Zawinski and others at Lucid developed Lucid Emacs, based on an early version of Gnu Emacs 19. Due to various complications, the code bases were never merged. This later became XEmacs.

Basic concepts

  • The basic editing unit of Emacs is a buffer.
  • Each buffer has associated with it a major mode and possibly several minor modes.
  • A major mode is a configuration for editing certain types of files, including highlighting, keybindings, formating, etc.
  • A minor mode is just added functionality for a buffer.

Basic keystrokes

  • You must absolutely know these keys to get started:
    • C-x C-f -- Open a file
    • C-x C-s -- Save a file
    • C-x C-c -- Quit Emacs
    • C-h t -- The Emacs Interactive Tutorial
    • M-x -- Run an Emacs command by name
  • Guide to key mnemonics
    • C -- Control
    • M -- Meta, which is either alt, esc, or sometimes the Windows key, depending on your configuration
    • A -- Alt

Basic Movement Commands

  • C-a, C-e -- Go to the beginning and end of line
  • M-v, C-v -- Page up, Page Down
  • C-f, M-f -- Forward character, forward word
  • C-b, M-b -- Backward character, backward word
  • C-p, C-n -- Up line, down line
  • ESC <number> -- repeat command number times
    • Example: ESC 15 C-n -- Go down 15 lines

Basic Editing commands

  • C-d, M-d -- Kill character forward, kill word forward
  • C-k -- Kill from point to remainder of line
  • C-y -- Yank the top of the kill ring
  • C-SPC -- Mark
  • C-w, M-w -- Cut marked region, copy marked region
  • M-% -- Query replace

Customing Emacs

  • By hand customization is done by editing ~/.emacs in Gnu Emacs or ~/.xemacs/init.el
  • For help in customization, do M-x customize-browse.
